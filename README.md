# EduPlatform

## Тема проекта
Образовательная платформа / курсы

## Описание
Проект представляет собой образовательный сайт с тёмной темой, фиолетово-синими акцентами и адаптивной версткой.  
Сайт содержит страницы для просмотра курсов, регистрации пользователей, просмотра профиля и контактов.  
Цель проекта — создать красивый и функциональный интерфейс для онлайн-образования.

## Страницы сайта
1. Главная (`index.html`)
2. Все курсы (`courses.html`)
3. Регистрация (`register.html`)
4. Профиль пользователя (`profile.html`)
5. Контакты (`contacts.html`)

## Используемые технологии
- HTML5
- CSS3 (Flexbox, Grid)
